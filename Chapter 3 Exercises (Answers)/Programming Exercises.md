# **Question 2**
**Code**:
```python
import math

diameter = float(input("Enter the diameter of the pizza (inches): "))
price = float(input("Enter the price of the pizza ($): "))
radius = diameter / 2
area = math.pi * radius**2
cost_per_sqin = price / area

print("Cost per square inch:", cost_per_sqin)
```
